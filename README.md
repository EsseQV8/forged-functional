# Ignition

A personal motivation app. Opens to a quote on a fire-toned background.

- Feed the Fire = next quote
- More / Less = tune how often a quote appears
- Gas tab = links to watch/listen (video, music)
- Flame tab = links to read (social, poems, written word)

Quotes and links sync across devices via Firebase.
To add quotes: edit the QUOTES list near the top of index.html.
